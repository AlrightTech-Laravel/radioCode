How to Enter your Nissan Radio Code

Find how to enter your Nissan radio code in vehicles such as Juke, Qashqai, Micra, Note and other vehicles.

How To Enter Nissan Connect Radio Code

  • When turned on, the radio will say system locked, enter the pin, press OK.
  • Numbers 0-9 will display across the top, enter your 4 digit code and press OK.
  • Your navigation system may display a message, just press accept, then your radio will come on.

Helpful Information
  • 3 tries is a wait 1 hour
  • 6 tries is a wait 12 hour
  • 9 tries is a wait 24 hour
  • If you enter your code 10 times, your radio will lock, forcing you to go to a dealer to get it unlocked.

 

How To Enter Nissan Daewoo Radio Code

  • When the radio is turned on 'CODE IN' should display on the radios display to input:
  • Repeatedly press button 1 until the correct first digit of the code is displayed.
  • Repeatedly press button 2 until the correct second digit of the code is displayed.
  • Repeatedly press button 3 until the correct third digit of the code is displayed.
  • Repeatedly press button 4 until the correct fourth digit of the code is displayed.
  • If you have entered the code correctly on the radios display, simply press enter.
  • To confirm the code you may need to press and hold the 6, or TA button for 10-15 seconds.
  • Your radio may turn off, turn it back on and the radio will be working.

How To Enter Nissan Clarion Radio Code

  • When turned on, the radio will display CODE IN.
  • Press button 1, and then CODE 0000 will appear, allowing you to enter the code.
  • Repeatedly press button 1 until the correct first digit of the code is displayed.
  • Repeatedly press button 2 until the correct second digit of the code is displayed.
  • Repeatedly press button 3 until the correct third digit of the code is displayed.
  • Repeatedly press button 4 until the correct fourth digit of the code is displayed.
  • Once entered, press the TUNE SEARCH button, and CODE OK will appear, prompting your radio to come on.

How To Enter Nissan 6CD Blaupunkt Radio Code

  • When turned on, CODE will be displayed.
  • Repeatedly press button 1 until the correct first digit of the code is displayed.
  • Repeatedly press button 2 until the correct second digit of the code is displayed.
  • Repeatedly press button 3 until the correct third digit of the code is displayed.
  • Repeatedly press button 4 until the correct fourth digit of the code is displayed.
  • Once you have entered the code, press the TA button, prompting your radio to come on.

How To Enter Nissan K12 Blaupunkt Radio Code

  • When turned on, CODE will be displayed, followed by 0000.
  • Repeatedly press button 1 until the correct first digit of the code is displayed.
  • Repeatedly press button 2 until the correct second digit of the code is displayed.
  • Repeatedly press button 3 until the correct third digit of the code is displayed.
  • Repeatedly press button 4 until the correct fourth digit of the code is displayed.
  • After you have entered the code, press and hold the ^ button for a few seconds, prompting your radio to come on.
